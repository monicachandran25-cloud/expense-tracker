# 💰 Expense Tracker (Python)

A simple Python application to track daily expenses and generate reports with category-wise analysis.

---

## 🚀 Features

- Add daily expenses
- Categorize spending
- Generate reports
- Visualize expenses with charts

---

## 🧰 Tech Stack

- Python
- Pandas
- Matplotlib

---

## ▶️ Usage

```bash
python main.py
